The BioChrome Atlas

Experiments in Natural Dyes. An interactive way to explore natural-dye colour swatches, and to rebuild images using those swatches as tiles.

Built as an extension of my Fabricademy program coursework to analyse the dyed swatches made during BioChromes week.

This set of tools is also a digital twin of what is intended to be a Material Swatches library. While these tools relate primarily to an analysis and play with colour, the Material library will encompass much more. Consequently, other tools will also turn up here.
