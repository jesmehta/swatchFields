The Swatch Atlas

Experiments in Natural Dyes and BioChromes. An interactive way to explore natural-dye colour swatches, and to rebuild images using those swatches as tiles.


Built as an extension of my Fabricademy program coursework to analyse the dyed swatches made during BioChromes week.


This set of tools is also a digital twin of what is intended to be a Material Swatches library. While these tools relate primarily to an analysis and paly with colour, the Material library will encompass much more. Consequently, other tools will also turn up here.

Swatch Explorer (Polar HSB & Grid)

Explore the entire swatch dataset visually through a colour wheel or parameter grid.

  • Load swatches automatically
  • Switch between Polar and Grid views
  • Filter by dyestuff, pH, mordant, additive and dye time
  • Hover on any swatch to see a large preview and metadata
  • Zoom and adjust swatch size for clarity
Open Swatch Explorer →

Swatch Mosaic Generator

Load an image and re-create it using swatches as pixels — each tile maps to one unique swatch.

  • Load an image (upload or URL)
  • Choose tile size
  • Adjust H/S/B match weights for character or accuracy
  • Hover to inspect which swatch each tile came from
  • Export the mosaic as PNG and a tile→swatch CSV
