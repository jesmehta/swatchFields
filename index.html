Dyestuff Swatch Tools

Visualise and recompose images using a library of dyed swatches. Built for experiments, teaching, and fun.

Polar / Grid Visualisation (v2.3)

Explore the swatch set on a colour wheel (HSB) or as a sortable/filterable grid. Hover to inspect a swatch with full metadata.

Open Polar / Grid

Hardcoded folders: imagesBordered/ (display) and imagesSuperCrop/ (hover preview). Lookup: swatch_lookup.json.

Swatch Mosaic (v1.1)

Recreate any image using only your swatches (one tile = one swatch). Exports a traceable CSV mapping “tile → swatch”.

Open Mosaic

Hardcoded folders: imagesSuperCrop/. Lookup: swatch_lookup.json.

Instructions

  • Put swatch images in imagesBordered/ (framed) and imagesSuperCrop/ (tight squares).
  • Place swatch_lookup.json at the site root (or inside each tool folder if you prefer; update paths accordingly).
  • Host on GitHub Pages (this is pure static HTML + JS).

Made for rapid exploration. A future v3.0 will add D3 for cleaner axes/zoom and Pixi/Canvas for bigger datasets.
